## Sensors
Your board supports the following sensors:
- 4 electrochemical sensors
- 1 wind speed sensor
- 1 I2C particulate matter sensor
- 1 DHT11/22 sensor

## Powering the board
Your board supports the following energy sources:
- Solar panel
- Battery
- USB-C
